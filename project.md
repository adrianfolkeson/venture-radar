 Setup Supabase                                                                                                                                                                                                
                                                                                                                                                                                                               
 Kör denna SQL i Supabase SQL Editor:                                                                                                                                                                          
                                                                                                                                                                                                               
 ```sql                                                                                                                                                                                                        
   CREATE TABLE opportunities (                                                                                                                                                                                
     id UUID PRIMARY KEY DEFAULT gen_random_uuid(),                                                                                                                                                            
     name TEXT NOT NULL,                                                                                                                                                                                       
     description TEXT,                                                                                                                                                                                         
     source TEXT NOT NULL,                                                                                                                                                                                     
     source_url TEXT,                                                                                                                                                                                          
     mentions_w1 INT DEFAULT 0,                                                                                                                                                                                
     mentions_w2 INT DEFAULT 0,                                                                                                                                                                                
     mentions_w3 INT DEFAULT 0,                                                                                                                                                                                
     mentions_w4 INT DEFAULT 0,                                                                                                                                                                                
     upvotes INT DEFAULT 0,                                                                                                                                                                                    
     comments INT DEFAULT 0,                                                                                                                                                                                   
     avg_sentiment DECIMAL(3,2) DEFAULT 0,                                                                                                                                                                     
     ai_summary TEXT,                                                                                                                                                                                          
     ai_pain_points TEXT[],                                                                                                                                                                                    
     ai_competition_level TEXT,                                                                                                                                                                                
     ai_should_build BOOLEAN,                                                                                                                                                                                  
     velocity_score INT DEFAULT 0,                                                                                                                                                                             
     saturation_score INT DEFAULT 0,                                                                                                                                                                           
     monetization_score INT DEFAULT 0,                                                                                                                                                                         
     timing_score INT DEFAULT 0,                                                                                                                                                                               
     overall_score INT DEFAULT 0,                                                                                                                                                                              
     tags TEXT[],                                                                                                                                                                                              
     user_rating INT,                                                                                                                                                                                          
     user_notes TEXT,                                                                                                                                                                                          
     marked_built BOOLEAN DEFAULT FALSE,                                                                                                                                                                       
     first_seen_at TIMESTAMP DEFAULT NOW(),                                                                                                                                                                    
     status TEXT DEFAULT 'active'                                                                                                                                                                              
   );                                                                                                                                                                                                          
   CREATE INDEX idx_score ON opportunities(overall_score DESC);                                                                                                                                                
 ```

 Scoring                                                                                                                                                                                                       
                                                                                                                                                                                                               
 ```                                                                                                                                                                                                           
   OVERALL = velocity×0.35 + (100-saturation)×0.25 + monetization×0.25 + timing×0.15                                                                                                                           
 ```                                                                                                                                                                                                           
                                                                                                                                                                                                               
 - Velocity (35%): Tillväxt de senaste veckorna                                                                                                                                                                
 - Saturation (25%): Hur mycket konkurrens (lägre = bättre)                                                                                                                                                    
 - Monetization (25%): Betalningsvilja                                                                                                                                                                         
 - Timing (15%): Är det rätt timing?
